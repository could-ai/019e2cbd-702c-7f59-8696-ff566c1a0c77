# USD.AI

USD.AI is an AI-powered cryptocurrency prediction dashboard built with Flutter. It provides real-time-like market overviews, AI confidence scores, and trend predictions for various cryptocurrencies.

## Features
- **Market Overview**: View current prices, 24h changes, and market trends.
- **AI Predictions**: See AI-generated confidence scores and price predictions for popular cryptocurrencies.
- **Dark Theme**: A sleek, modern dark interface designed for financial tools.
- **Cross-Platform**: Fully responsive design that works seamlessly on iOS, Android, Web, and Desktop.

## Tech Stack
- Flutter
- Dart

## Getting Started

To run the project locally, ensure you have Flutter installed.

```bash
flutter pub get
flutter run
```
